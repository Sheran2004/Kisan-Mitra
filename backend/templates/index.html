Kaise Kaam Karta Hai?

Technology explained in simple words

1️⃣ Crop Recommendation
Random Forest Classifier (300 trees) use kiya gaya hai. 10,000 samples pe train hua — 20 crops ke ideal soil aur climate conditions ke saath. Accuracy: 94.8%
2️⃣ Disease Detection
Patte ki image se RGB color features extract kiye jaate hain. Brown ratio, yellow ratio, R/G/B means — yeh sab disease indicators hain. Random Forest model 12 diseases identify karta hai. Accuracy: 91.5%
3️⃣ Flask REST API
Backend Flask (Python) pe chal raha hai — localhost:5000. HTML frontend seedha API call karta hai aur real predictions leta hai.
